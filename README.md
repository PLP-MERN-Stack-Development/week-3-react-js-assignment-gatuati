Prerequisites
- Node.js v18 or later
- npm (v9+) or pnpm (v8+)
  
Project Structure

- **React** for UI
- **Vite** as the build tool
- **Tailwind CSS v4** for styling
- Routing via `react-router-dom`
- Context API for theme management (light/dark)
